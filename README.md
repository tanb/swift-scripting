# swift-scripting

## Requisites
Cocoapods (install via gem)
Cocoapods-Rome (install via gem)

## Makefile
Switch Swift2.0
```
$ make switch-beta
```

Switch current version
```
$ make switch-current
```

Run Swift REPL
```
$ make shell
```

Build src directory swift script
```
$ make build-swift
```

Build and run
```
$ make run
```

Run swift script
```
$ make runscript SCRIPT_FILE="csv_script.swift"
```
